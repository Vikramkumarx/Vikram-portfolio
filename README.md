## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: Local server (Live Server, http-server, etc.)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Vikramkumarx/portfolio.git
   cd portfolio
   ```

2. **Open the website**
   - Simply open `index.html` in your browser
   - Or use a local server:
     ```bash
     # Using Python
     python -m http.server 8080
     
     # Using Node.js
     npx http-server -p 8080
     ```

3. **View in browser**
   - Navigate to `http://localhost:8080`

## ğŸ“ Customization

### Update Personal Information

1. **Edit `index.html`**:
   - Update name, title, bio in the hero section
   - Modify skills, projects, and contact information
   - Add/remove sections as needed

2. **Edit `style.css`**:
   - Change color scheme in `:root` variables
   - Adjust spacing, fonts, and animations
   - Customize responsive breakpoints

3. **Edit `script.js`**:
   - Modify typing effect texts
   - Update form submission logic
   - Add custom interactions

### Add Your Resume

Replace `assets/Vikram_Kumar_Resume.pdf` with your own resume file.

### Update Images

Replace images in the `assets/` folder:
- `profile.png` - Your profile picture
- `about.png` - About section image
- `project1.jpg` to `project6.jpg` - Project screenshots
